# 2026/4/5 走遍德国第一单元学习笔记

## 前言

《走遍德国》(Menschen) 是由 Hueber 出版社发行的德语学习教材，因其内容实用、配套完善被广泛使用。第一单元通常涵盖**问候与自我介绍**，是德语学习的起点。本文将第一单元的核心内容整理成笔记，适合A1水平学习者参考。

### 1.4 人称代词

| 人称 | 主格（第一格） | 宾格（第四格） | 中文 |
|------|---------------|---------------|------|
| 我 | ich | mich | I / me |
| 你 | du | dich | you |
| 他 | er | ihn | he / him |
| 她 | sie | sie | she / her |
| 它 | es | es | it |
| 我们 | wir | uns | we / us |
| 你们 | ihr | euch | you (pl.) |
| 您 | Sie | Sie | you (formal) |
| 他们/她们 | sie | sie | they / them |

> **重要**：德语中"您 Sie"（大写）是正式称呼，用于陌生人或长辈；"你 du"用于朋友、家人、孩子。

## 二、核心句型

### 2.1 问候与告别

```
Guten Morgen! — 早上好！（对方回应：Guten Morgen!）
Hallo! — 你好！
Wie geht es Ihnen? — 您好吗？（正式）
Wie geht es dir? — 你好吗？（非正式）
→ Gut, danke. / Nicht schlecht. / Es geht. — 挺好的，谢谢。/ 还行。

Tschüss! / Ciao! — 再见！（非正式）
Auf Wiedersehen! — 再见！（正式）
Tschüss, bis morgen! — 明天见！
```

### 2.2 自我介绍

```
Ich heiße... — 我的名字是...
Mein Name ist... — 我的名字是...
Ich bin... — 我是...
Ich komme aus... — 我来自...
Ich wohne in... — 我住在...

完整示例：
Hallo! Ich heiße Zhang Wei. Ich komme aus Beijing. Ich wohne in Berlin.
你好！我叫张伟。我来自北京。我住在柏林。
```

### 2.3 询问信息

```
Wie heißen Sie? — 您叫什么名字？（正式）
Wie heißt du? — 你叫什么名字？（非正式）
Woher kommen Sie? — 您从哪里来？（正式）
Woher kommst du? — 你从哪里来？（非正式）
Wo wohnen Sie? — 您住在哪里？（正式）
Wo wohnst du? — 你住在哪里？（非正式）

Wie alt sind Sie? — 您多大了？（正式）
Wie alt bist du? — 你多大了？（非正式）
```

### 2.4 数字相关表达

```
Was kostet das? — 这个多少钱？
Das kostet zwanzig Euro. — 这个20欧元。
Ich möchte drei Brötchen. — 我想要三个小面包。

电话号码表达（德语逐数字读）：
0123 456789 → null eins zwei drei, vier fünf sechs sieben acht neun
```

## 三、基础语法

### 3.1 德语动词变位（现在时）

德语动词根据主语人称变化，称为**动词变位**。以"heißen"为例：

| 人称 | 变位 | 词尾 |
|------|------|------|
| ich | heiße | -e |
| du | heißt | -st |
| er/sie/es | heißt | -t |
| wir | heißen | -en |
| ihr | heißt | -t |
| Sie | heißen | -en |
| sie | heißen | -en |

**规则动词变位规律**：
- 大部分动词：词干 + 对应词尾
- 以 -t, -d, -chn, -sch, -gn, -ff 结尾的动词，第二人称单数加 -est

```
arbeiten (工作):
du arbeitest, er/sie/es arbeitet, ihr arbeitet

heißen (叫作):
du heißt, er/sie/es heißt, ihr heißt

sprechen (说话):
du sprichst, er/sie/es spricht, ihr sprecht  ⚠️ 强变化动词
```

### 3.2 常见动词变位表

| 原形 | ich | du | er/sie/es | wir | ihr | Sie/sie |
|------|-----|----|-----------|-----|------|----------|
| sein (是) | bin | bist | ist | sind | seid | sind |
| haben (有) | habe | hast | hat | haben | habt | haben |
| heißen | heiße | heißt | heißt | heißen | heißt | heißen |
| kommen | komme | kommst | kommt | kommen | kommt | kommen |
| wohnen | wohne | wohnst | wohnt | wohnen | wohnt | wohnen |
| sprechen | spreche | sprichst | spricht | sprechen | sprecht | sprechen |

### 3.3 句子结构

**德语基本语序**：
```
陈述句： 主语 + 动词 + 其他
Ich     heiße  Zhang Wei.
Ich     komme  aus China.

疑问句： 动词 + 主语 + 其他
Heißen Sie so?
Kommen Sie aus China?

特殊疑问句： 疑问词 + 动词 + 主语 + 其他
Wie       heißen Sie?
Woher     kommen Sie?
Was       machen Sie?
```

**位置框架**（动词在第二位）：
```
Ich          sehe   heute    meinen Freund.      ✅
Heute        sehe   ich       meinen Freund.      ✅
Ich          sehe   meinen Freund    heute.      ✅
Ich          sehe          meinen Freund heute.  ❌（动词分开）
```

### 3.4 名词性数格

**德语名词的性**：
- der（阳性）、die（阴性）、das（中性）
- 没有简单规律，需要**死记硬背**

| 单词 | 词性 | 中文 |
|------|------|------|
| der Mann | 阳性 | 男人 |
| die Frau | 阴性 | 女人 |
| das Kind | 中性 | 孩子 |
| der Name | 阳性 | 名字 |
| die Stadt | 阴性 | 城市 |
| das Land | 中性 | 国家 |

> **记忆技巧**：用 "der" / "die" / "das" 配合背单词，如 der Name，而不是只背 Name。

**名词的复数**：

| 原形 | 复数形式 | 中文 |
|------|---------|------|
| der Name | die Namen | 名字们 |
| die Frau | die Frauen | 女人们 |
| das Kind | die Kinder | 孩子们 |
| der Mann | die Männer | 男人们 |
| die Stadt | die Städte | 城市们 |

## 四、会话模板

### 4.1 初次见面

```
A: Guten Tag! Ich heiße Anna. Wie heißen Sie?
   你好！我叫Anna。您叫什么名字？

B: Guten Tag! Ich heiße Li Ming. Freut mich!
   你好！我叫李明。很高兴认识你！

A: Freut mich auch! Woher kommen Sie?
   我也很高兴！您来自哪里？

B: Ich komme aus China. Und Sie?
   我来自中国。您呢？

A: Ich komme aus Deutschland.
   我来自德国。

B: Wo wohnen Sie in Deutschland?
   您在德国住在哪里？

A: Ich wohne in Berlin. Und Sie?
   我住在柏林。您呢？

B: Ich wohne zur Zeit in München.
   我目前住在慕尼黑。

A: Sehr nett! Auf Wiedersehen!
   很高兴认识您！再见！

B: Auf Wiedersehen!
   再见！
```

### 4.2 商店购物

```
Kunde: Guten Tag! Was kostet das?
顾客：您好！这个多少钱？

Verkäufer: Das kostet zehn Euro.
售货员：这个10欧元。

Kunde: Und dieses hier?
顾客：这个呢？

Verkäufer: Dieses kostet fünfzehn Euro.
售货员：这个15欧元。

Kunde: Ich nehme das. Das macht zusammen...
顾客：我要这个。一共是...

Verkäufer: Fünfundzwanzig Euro, bitte.
售货员：25欧元。

Kunde: Hier bitte. Danke!
顾客：给您。谢谢！

Verkäufer: Bitte schön! Auf Wiedersehen!
售货员：不客气！再见！
```

## 五、学习重点与常见错误

### 5.1 必须掌握的要点

```
✅ 1. 人称代词六个形式（ich, du, er/sie/es, wir, ihr, Sie/sie）
✅ 2. 规则动词变位（-e, -st, -t, -en, -t, -en）
✅ 3. 数字0-20的读写
✅ 4. 基本问候语的使用场景
✅ 5. Wie heißen Sie? / Woher kommen Sie? 等问句
✅ 6. 动词在句子第二位的原则
```

### 5.2 常见错误

```
❌ 错误：Ich heiße Zhang und ich kommen aus China.
✅ 正确：Ich heiße Zhang und ich komme aus China.
       （主语是ich，动词用komme，不是kommen）

❌ 错误：Ich heiße li Ming.
✅ 正确：Ich heiße Li Ming.
       （名字首字母大写）

❌ 错误：Wie heißt du? → Ich heiße er Li.
✅ 正确：Wie heißt du? → Ich heiße Li (Ming).
       （自我介绍不说"我是他"，直接说名字）

❌ 错误：Ich komme von China.
✅ 正确：Ich komme aus China.
       （来自某地用aus，不是von）

❌ 错误：Guten Morgen, Li Ming. (对陌生人早上用)
✅ 正确：Guten Tag, Herr/Frau + 姓
       （早上好可用于任何人，但有头衔要用正式称呼）
```

### 5.3 发音注意事项

```
德语特殊发音：
ä → 类似"诶"
ö → 类似"约"（圆嘴）
ü → 类似"鱼"（圆嘴）
ß → ss 的发音
ch (a/o/u后) → 类似于h
ch (e/i后) → 类似于"西"
r → 小舌音（初期可用h代替）

常见错误：
- "ich" 不要发成"一区"，而是"衣区"
- "acht" 的ch要发出来，不是"阿特"
- "fünf" 的ü要圆唇
```

## 六、练习题

### 6.1 选择题

```
1. Wie _____ Sie?
   a) heiße    b) heißt    c) heißen
   答案：c) heißen（正式问句，动词提前）

2. Ich _____ aus Japan.
   a) kommst    b) kommen    c) kommt
   答案：c) kommen（ich用komme，但这里要选原形kommt是错的！
   实际答案：b) kommen，因为Ich + kommen）

3. "Guten Abend" 用于：
   a) 早上    b) 下午    c) 晚上
   答案：c) 晚上

4. 12的德语是：
   a) zwölf    b) zwanzig    c) zwei
   答案：a) zwölf
```

### 6.2 连线题

```
Guten Morgen          a) 告别
Tschüss               b) 下午问候
Guten Tag             c) 早上问候
Wie geht es Ihnen?   d) 问名字
Wie heißen du?        e) 问健康
答案：c, a, b, e, d
```

### 6.3 翻译练习（参考答案）

```
1. 你叫什么名字？
   Wie heißt du?

2. 我来自中国。
   Ich komme aus China.

3. 这个多少钱？
   Was kostet das?

4. 我很好，谢谢。
   Mir geht es gut, danke.

5. 明天见！
   Bis morgen!
```

```

□ 能够询问对方的名字、来自哪里、住在哪里
□ 能够用德语做简单的自我介绍

□ 掌握六个基本人称代词
□ 能够对规则动词进行现在时变位
□ 了解德语基本句型语序
□ 知道德语名词有性（der/die/das）
□ 能够进行简单的数字相关对话
```
